Change Tier

Change Tier upgrades and/or downgrades the demo product's tier based on the patient's desires. Whenever a demo hearing aid is connected and read, if Pro Fit identifies that a product tier upgrade and/or downgrade is available, a Change Tier option appears via Connect. The catalog of demo hearing aids and tiers available for updates are based on the Pro Fit edition and language selected via Configuration Preferences.

 

Demo Product Tier Updates

 

DemoProduct tier upgrades and/or downgrades are only available within the same product family. Based on Pro Fit's current configuration determines whether products can be upgraded and/or downgraded (only the receiver aid can be upgraded and/or downgraded for CROS/BiCROS products). If upgrades only are supported, this feature is not available if the hearing aid's current product tier is flagship. If downgrades only are supported, this feature is not available if the hearing aid's current product tier is entry-level. If both upgrades and downgrades are supported, this feature is always available regardless of the hearing aid's current product tier.

 

DemoProduct tier upgrades and/or downgrades automatically retain the frequency response and all applicable settings (programs, feature settings, fitting formula, etc.) if there's a one-to-one match between product tiers. Any settings that don't match (e.g., user control, wireless, etc.) will be set to default settings.

  1. Connect and select the desired demo hearing aid(s) to upgrade and/or downgrade via Connect.
  2. Click Change Tier and product information display.
    Note:  This option is only available if a product tier upgrade and/or downgrade is available for the connected hearing aid and the feature is supported based on Pro Fit's current configuration. The hearing aid's serial number and current product family and tier display. If the battery life is low or less than 25% for a rechargeable product, this option is disabled. The device must be recharged first in order to perform the upgrade and/or downgrade.
    Note:  If two demo hearing aids were selected, both aids will be updated simultaneously. Ear sides cannot be selected independently. If two hearing aids were connected but only one device can be upgraded and/or downgraded, only the one aid will be updated.
    Tip:  Click Close to cancel the product upgrade and/or downgrade.
  3. Select the desired product tier from the drop-down list.
    Note:  The tiers available are based on the Pro Fit edition and language selected via Configuration Preferences. If two demo hearing aids were connected and upgrades and/or downgrades are available for both devices, the product tier selected is automatically chosen for both aids. If different product tiers are desired for each ear side, click Close, and update each device separately.
  4. Click Change Tier and a progress bar indicates the overall status of completion.
    Note:  This button is only enabled if a new product tier was selected.
  5. Click Close once the product upgrade and/or downgrade is complete and the Connect screen automatically displays.
