Update Firmware

Update Firmware updates the hearing aid's firmware to operate at a more advanced performance level, oftentimes making new features available. Firmware is the computer code residing in the hearing aid circuit that tells the hearing aid how to function. Whenever a hearing aid is connected and read, if Pro Fit identifies that a newer firmware version is available, an alert  displays on the hearing aid card and an Update Firmware option appears via Connect. If a firmware update is available, it's highly recommended you update the hearing aid(s) to ensure optimal performance. If the firmware is not updated, both the alert  and the Update Firmware option will continue to appear until the hearing aid(s) is updated.

 

If binaural hearing aids contain different firmware versions that are incompatible, the hearing aids cannot be fit until the hearing aid with the lower firmware version is updated.

  1. Connect and select the desired hearing aid(s) to update via Connect.
  2. Click Update Firmware and important information regarding the firmware update display.
    Note:  This option is only available if a firmware update was identified. The hearing aid's serial number, current firmware version, and the new update version display. If the battery life is low or less than 25% for a rechargeable product, this option is disabled. The device must be recharged first in order to perform the update.
    Note:  If two hearing aids were connected and both devices require firmware updates, both aids will be updated simultaneously. Ear sides cannot be selected independently. If two hearing aids were connected but only one device requires a firmware update, only the one aid will be updated.
    Tip:  Click Close to cancel updating the firmware.
  3. Click Update Firmware and a progress bar indicates the overall status of completion.
  4. Click Close once the update is complete and the Connect screen automatically displays.
